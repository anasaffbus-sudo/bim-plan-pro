# 💰 خطط التسعير — BIM Plan Pro

> **آخر تحديث:** 2026-05-12 · **العملة:** SAR (ريال سعودي) · **الإطلاق الأوّل**
>
> القاعدة الرياضية المتّبعة (Day 4 - الجولة 4):
> - الوسطى = 3-4× الأولى
> - الأخيرة = 2.5× الوسطى على الأقلّ
> - الوسطى = 10-20% من القيمة الموفَّرة

## 📊 السياق السعري

| البند | القيمة |
|---|---|
| البديل الحالي (استشاري BIM لكتابة BEP) | 3,000 – 5,000 SAR / مشروع |
| القيمة الموفَّرة من المنتج | ~4,000 SAR لكل BEP (مال + أسابيع عمل) |
| الفائدة بالأرقام | يختصر 2-3 أسابيع كتابة وتنسيق → 30 دقيقة |
| الشريحة المستهدفة | BIM Managers · مقاولين · مكاتب استشارية صغيرة-متوسطة |

## 🎯 الخطط الثلاث

### 1️⃣ خطّة «المبتدئ» — للتجربة الجادّة
- **السعر:** `99 SAR / شهر` · أو `950 SAR / سنة` (وفّر 20%)
- **العميل المثالي:** مهندس BIM فردي أو مشروع تخرّج يحتاج خطة BEP واحدة بمواصفات ISO.
- **يشمل:**
  1. مشروع واحد فعّال في كل وقت
  2. توليد BEP كامل بـ AR/EN
  3. تصدير PDF + DOCX
  4. قوالب ISO 19650 الأساسية
  5. تخزين سحابي للمشروع

### 2️⃣ خطّة «المحترف» ★ الأكثر اختياراً
- **السعر:** `349 SAR / شهر` · أو `3,350 SAR / سنة` (وفّر 20%)
- **العميل المثالي:** مكتب استشاري أو مقاول يدير 3-10 مشاريع BIM متزامنة ويحتاج توحيد القوالب وسرعة التسليم.
- **يشمل:**
  1. حتى **10 مشاريع** فعّالة
  2. كل ميزات «المبتدئ»
  3. **شعار الشركة** على كل BEP (Branding)
  4. مكتبة قوالب موسّعة (LOD/EIR/MIDP)
  5. تصدير بصيغ متعدّدة + مشاركة مع الفريق

### 3️⃣ خطّة «المؤسّسي» — للشركات
- **السعر:** `999 SAR / شهر` · أو `9,590 SAR / سنة` (وفّر 20%)
- **العميل المثالي:** شركة كبيرة أو جهة حكومية تحتاج عدد مشاريع غير محدود + تخصيص كامل + دعم مباشر.
- **يشمل:**
  1. **مشاريع غير محدودة** + فريق غير محدود
  2. كل ميزات «المحترف»
  3. تخصيص القوالب بهوية الشركة
  4. دعم مباشر عبر WhatsApp + اجتماع تأهيل
  5. تكامل API (قيد التطوير) + لوحة إدارة المشرف
